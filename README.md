## Project Objective

The goal of the assignment was to:

- Understand user behavior from raw session data

## Dataset Overview

The dataset captures **online shopping session behavior** for an e-commerce platform. Each row represents a single user session and includes:

- Page views and time spent across different page types  
- Traffic source and visitor type (new vs. returning)  
- Temporal features such as month and seasonality  
- A binary outcome indicating whether a purchase occurred  

---

## Key Takeaways

- **User engagement matters:** Sessions with higher engagement (more pages viewed and longer duration) were significantly more likely to result in a purchase.  
- **Returning visitors drive revenue:** Returning users generated the majority of transactions, highlighting the value of retention and repeat engagement.  
- **Strong seasonality exists:** Purchase behavior varies by month, with clear peaks during high-traffic retail periods.  
- **Not all traffic is equal:** Certain traffic sources contributed disproportionately to revenue, suggesting opportunities for marketing optimization.  
- **Behavioral signals outperform calendar events:** Engagement metrics were more predictive of purchases than proximity to special promotional days.
